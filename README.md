# Assignment 6: Multilayer Neural Network with Hyperparameter Optimisation

Este repositorio contiene el trabajo realizado en conjunto con mis compañeros Sergio Alejandro Zamora Dávila y Perla Yazmín Antonio Pérez. 

En este proyecto se desarrolla una red neuronal multicapa con optimización de hiperparámetros aplicada a un problema de clasificación multiclase.

- `assignment.ipynb` - Desarrollo completo del trabajo  
- `assignment.html` - Versión en HTML del notebook

## Información del dataset

El dataset seleccionado para este proyecto es *Wine Recognition Data* (Forina et al., 1991), recuperado de [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/109/wine) Este conjunto de datos pertenece al área de análisis químico de alimentos y se utiliza para la clasificación de vinos italianos.

El propósito principal es clasificar los vinos en tres cultivares (variedades) distintos, basándose en su composición química. Todas las muestras provienen de la misma región de Italia.

## Características del dataset:
- Total de muestras: 178  
- Número de columnas: 14  
  - 13 variables independientes (características químicas)  
  - 1 variable objetivo (Class)  
- Clases: 3 (tipo de vino)

## Uso del notebook

Para utilizar correctamente este trabajo:

1. Descarga el archivo `assignment.ipynb`
2. Descarga el dataset en formato `.csv`
3. Coloca ambos archivos en la misma carpeta

Esto es importante para que, al ejecutar el notebook, el programa pueda acceder correctamente a los datos y cargar el dataset sin errores.
